# AI-Sentiment-Data-Analytics-Project

## How to run
1. Clone the repository to your local machine
2. Unzip `data.zip`, and ensure the `data` folder is in the same folder location as `comp3800f25_project.pynb`

File structure should look like the following


```
AI-Sentiment-Data-Analytics-Project/
├── data/
│   ├── COMP3800F25S01RawData.csv
│   └── COMP3800F25S01Keywords.csv
├── comp3800f25_project.ipynb
└── README.md
```
